The purpose of this web app is to demonstarte the complete MVC request flow
#How it works
The user will enter their name, favorite food, and age via the index.jsp file, the respone is sent to the processServlet.java which 
process and stores the data in reuqest attributes. Then the processServlet fowards the request to result.jsp file where it displays the result to the 
user using Expression Language (EL)
