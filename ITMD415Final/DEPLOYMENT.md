In order to deploy to your payara server you need to make sure your project is built using Eclipse Temurin JDK 21 (LTS), MySQL server, Intellij and a payara serever 6.x Community. 
To quicly test the app use the built in run button in Intellij to deploy your war Artifact and run your app, then go to the editEmployee page and edit an employees salary or department.
