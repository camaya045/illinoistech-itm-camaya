This app uses ADMIN and USER for the roles, the ADMIN is able to view and edit all employees along with their salary, tittle, and department. The user is only able to view all 
employees and their departments. The editEmployees.xhtml file is protected and only the admin is able to access it, with users being denied access. In order to test this, first 
deploy the app, and then try accessing the edit employee page using the users log in information when prompted and then attempt to log in using the admin log in information.
